# AO3 Envelope Work Skin

Copy-paste AO3 envelope effects built with work skins, limited HTML, and no
JavaScript.

## Use This Repo In 3 Steps

1. Read `guides/START-HERE-ZH.md`
2. Copy from `bundle/`
3. If something fails, read `guides/FAQ-ZH.md`

## User-Facing Files

- `bundle/work-skin.css`
- `bundle/smoke-test.html`
- `bundle/hover-template-zh.html`
- `bundle/tap-template-zh.html`
- `guides/START-HERE-ZH.md`
- `guides/FAQ-ZH.md`

## Developer-Only Files

- `dev/AO3-COMPATIBILITY.md`
- `dev/index.html`
- `dev/validation.html`
- `dev/scripts/ao3-cdp.mjs`
- `dev/scripts/capture-validation.sh`

These are for local preview and maintenance, not for AO3 posting.

## Core Constraints

- AO3 does not allow JavaScript in posted works.
- AO3 users can disable work skins.
- Downloads do not retain work skins.

The effect must stay optional and readable.
