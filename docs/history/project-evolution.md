# Project Evolution

This repository started as a single envelope-style AO3 work skin and later became a broader effects toolkit.

## Milestones

1. A first envelope prototype established the core trifold interaction pattern.
2. AO3 posting and validation notes were added once real platform constraints became clearer.
3. The repository shifted from one bundled effect to a toolkit structure with dedicated effect directories.
4. Additional effects were added: chat messages, polaroid, secret divider, and typewriter.
5. Shared preview scaffolding and demo media were introduced for local review.
6. The public repository was cleaned up around `effects/`, demo assets, and maintainer docs.
7. The repository was later simplified again around a four-file effect contract:
   - `work-skin.css`
   - `example.html`
   - `preview.html`
   - `guide.md`
8. The old multi-template effect structure, four-panel preview matrix, and broad committed verification script were removed in favor of a smaller preview shell and GIF-only tooling.
